# Resuming: Mood Sentence Generation Task

## What this is

The app `conjugations.json` needs example sentences for verb+person combinations
where **subjonctif présent** and **indicatif présent** produce different conjugated forms.

## Current status

- **16 prompt files** already generated (`scripts/mood_prompt_01_of_16.txt` through `...16_of_16.txt`)
- Each prompt file contains the exact verb+person combinations that still need sentences
- A **merge script** is ready: `scripts/merge_mood_results.py`
- Target file: `app/src/main/assets/conjugations.json`

## What needs to be done

### Step 1: Generate result files

For each prompt file `mood_prompt_XX_of_16.txt`, send the contents to a capable language model
and save the raw JSON response as `mood_result_XX.json` in the same `scripts/` directory.

The prompt files are self-contained — they include the system prompt, instructions, rules, and
the list of verb+person combinations to process.

Expected output format per prompt (example):
```json
{
  "balayer": {
    "subjonctif": {
      "3": {"fr": "Il faut que nous ___ le couloir.", "de": "Wir müssen den Korridor fegen.", "blank": "balayions"}
    },
    "present": {
      "3": {"fr": "Nous ___ la terrasse chaque matin.", "de": "Wir fegen die Terrasse jeden Morgen.", "blank": "balayons"}
    }
  }
}
```

Person index mapping: je=0, tu=1, il/elle=2, nous=3, vous=4, ils/elles=5

### Step 2: Run the merge script

```bash
cd scripts
python3 merge_mood_results.py
```

This will:
- Read all `mood_result_*.json` files
- Merge sentences into `conjugations.json` (only filling gaps, never overwriting)
- Print a summary of what was added

## Key rules for generation

- The conjugated form must appear **exactly** as listed in the `NEED: ...=form` field
- Replace the conjugated form with `___` (three underscores) in the `fr` sentence
- The `blank` field must equal the exact conjugated form
- Sentence length: 5–12 words
- Vary the trigger phrases for subjonctif (see prompt files for the full list)
- German translation must be natural
